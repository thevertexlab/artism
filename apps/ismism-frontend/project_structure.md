# Ismism Machine Project Structure Documentation

## Project Overview

The Ismism Machine is a visual timeline tool for exploring art movements and schools, helping users understand the relationships and development between different art movements. The project uses a front-end and back-end separated architecture, with React+TypeScript for the front-end and Node.js+Express+MongoDB for the back-end.

## Technology Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- Axios (HTTP Client)

### Backend
- Node.js
- Express
- MongoDB
- Mongoose
